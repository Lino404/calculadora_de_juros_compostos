# 🧮 Calculadora de Juros Compostos

Projeto acadêmico desenvolvido para a disciplina de Programação com o objetivo de simular a evolução de investimentos financeiros através do cálculo de juros compostos.

---

## 🚀 Funcionalidades
* **Fase 1 (Funcional):** Estrutura HTML básica e lógica matemática de cálculo em JavaScript.
* **Fase 2 (Estética):** Interface visual moderna, responsiva e estilizada via CSS.

---

## 🛠️ Tecnologias Utilizadas
* **HTML5** - Estrutura da página.
* **CSS3** - Design moderno e responsividade.
* **JavaScript (ES6)** - Lógica de cálculo dos juros.
* **Git & GitHub** - Versionamento e gerenciamento de releases (`v1.0.0` e `v2.0.0`).
